=============
IDWarp
=============

``IDWarp`` is a tool for performing mesh manipulation of 3-dimensional
multi-block, overset and unstructured meshes.

Limitations
===========

* Cannot be used where you have 2 symmetry planes e.g. in a 2D case. Use pywarp instead.
